# Multi-UAV Search & Rescue Carrier System

**TEKNOFEST 2025 — Proposal Stage (not built)**

## Overview

This project proposed a carrier UAV designed to deploy four sub-drones across a disaster area to support search and rescue operations. The carrier would house and release the sub-drones, which were intended to search for people in distress and support rescue coordination.

## Status

This project's technical proposal did not advance past TEKNOFEST's review stage, so it did not receive competition funding and was not physically built. The work in this repository reflects the design and engineering work completed up to that point.

## What's in this repository

- **Electrical system design** — completed circuit and power architecture for the sub-drone units
- **Embedded control software** — control logic for the sub-drone units
- **Mechanical concept design** — initial structural concept for the carrier UAV's housing and deployment mechanism
- **Design documentation** — technical proposal materials prepared for competition submission

## What we learned

During design review, the mechanical integration and deployment-mechanism reliability were identified as the project's critical risk area — this is the main reason the proposal didn't advance. The electrical and software components were completed and functioned as designed in isolated testing; the unresolved piece was reliably and safely releasing four sub-drones from a single carrier platform.
